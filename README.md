Dieses Repository bündelt praxisorientierte Security-Labs aus den Bereichen Web Security, API Security, System Exploitation und Secure Software Development.

Ich habe reale Schwachstellen wie IDOR, XSS, JWT-Manipulation, CORS-Misconfigurations, Remote Code Execution, HTTP Request Smuggling, File Upload Exploitation sowie API Excessive Data Exposure identifiziert und praktisch ausgenutzt. Zusätzlich habe ich Passwort-Hashes geknackt, Buffer-Overflow-Angriffe (Stack/Heap) durchgeführt und Netzwerkangriffe analysiert.

Parallel dazu habe ich sichere Webanwendungen und REST-Services entwickelt und abgesichert – inklusive RBAC, Access-Control-Absicherung, PBKDF2-Hashing, Login-Throttling, sicherer JWT-Verarbeitung und Hardening sensibler Daten.
