Dieses Repository dokumentiert die Labs aus Software & System Security 1 mit Fokus auf Web-Security, klassische Exploitation und Secure Coding.

Dabei wurden reale Schwachstellen wie IDOR, XSS, JWT-Manipulation, CORS-Misconfigurations, Remote Code Execution, HTTP Request Smuggling, File Upload Exploitation sowie API Excessive Data Exposure identifiziert und praktisch ausgenutzt. Zusätzlich wurden Passwort-Hashes geknackt, Buffer-Overflow-Angriffe (Stack/Heap) durchgeführt und Netzwerkangriffe analysiert.
